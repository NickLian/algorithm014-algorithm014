- 移动零

> 我的第一个版本，结果是执行时间超时

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int l = 0, f = 0;
        while (nums.length != 1 && f <= nums.length - 1) {
            if (nums[f] == 0) {
                f++;
                continue;
            }
            if (nums[l] == 0) {
                nums[l] = nums[f];
                nums[f] = 0;
                l++;
            }
        }
    }
}
```
